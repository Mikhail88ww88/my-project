# IISaller — AI-продавец для услуги поздравлений ростовыми куклами

## Что это

AI-консультант (продавец), который работает 24/7 на сайте компании.
Консультирует клиентов, называет цены, собирает номер телефона и передаёт лид менеджеру.
Основная ценность — обработка обращений вне рабочего времени (ночь, раннее утро, поздний вечер).

---

## Для кого

**Клиенты** — люди, которые хотят заказать поздравление ростовыми куклами. Часто пишут вне рабочего времени.

**Менеджеры компании** — получают в CRM готового лида с телефоном и историей переписки. Обрабатывают в рабочее время (06:00–20:00 МСК).

---

## Этап 1 — Что делаем сейчас (MVP)

### Виджет на сайте
- Кнопка чата в правом нижнем углу сайта
- Диалог прямо на сайте — без Telegram, WhatsApp и соцсетей
- Работает на Claude API

### Что умеет бот
- Консультирует по услуге: персонажи, описание, стоимость
- Называет цену с учётом города и выезда за город
- Работает 24/7, не упоминает рабочие часы и часовые пояса
- Вне рабочего времени отвечает: *"Передаю заявку специалисту, оставьте номер — свяжемся с вами"*
- Собирает номер телефона клиента

### Что происходит после получения телефона
- Сделка создаётся в AmoCRM (воронка "Горячие лиды")
- Полная история переписки прикрепляется к сделке
- Тебе приходит уведомление в Telegram с телефоном и диалогом
- В Яндекс Метрике срабатывает цель (как отправка формы)

### Незавершённые диалоги
- Все диалоги сохраняются в БД с первого сообщения
- Клиент написал но не оставил телефон → сделка в AmoCRM (воронка "Не дошли")
- Статусы диалогов: новый / в процессе / лид / ушёл

### Логика цен
- Базовая цена — 4 500₽ (все города кроме Москвы и СПб)
- Москва и СПб — отдельная цена (уточняется)
- Выезд за город — доплата по километражу от центра (уточняется)
- Бот называет примерную цену, финальную подтверждает менеджер

### UTM-метки
- Бот фиксирует с какой рекламы пришёл клиент
- Передаётся в AmoCRM вместе с лидом

## Этап 2 — Планируется позже

- Подключение Telegram-бота
- Подключение ВКонтакте
- Подключение WhatsApp
- Подключение Макс (VK Messenger) — под вопросом
- Подключение Авито — под вопросом
- Битрикс24 (вторая CRM параллельно с AmoCRM)
- Яндекс Директ API — аналитика расходов по кампаниям
- Яндекс Метрика API — данные по трафику и поведению
- Сквозная аналитика: реклама → бот → лид → сделка → выручка
- AI-аналитик рекламы: еженедельные отчёты с рекомендациями
- Учёт токенов по клиентам (для продажи бота другим бизнесам)

## Технологии

- **AI** — Claude API (Anthropic)
- **Бэкенд** — Python или Node.js
- **База данных** — PostgreSQL
- **CRM** — AmoCRM
- **Аналитика** — Яндекс Метрика (цели)
- **Уведомления** — Telegram Bot API
- **Хостинг** — VPS сервер (~300–500₽/мес)
