# Правила работы с проектом

## Структура файлов
- Каждая страница — один HTML-файл, всё внутри (HTML + CSS + JS)
- Никаких внешних библиотек и фреймворков
- Никаких отдельных .css и .js файлов — только если явно попросили

## Дизайн
- Фон — тёмный
- Текст — светлый
- Акцентный цвет — оранжевый
- Адаптивная вёрстка: всё должно нормально выглядеть на телефоне

## Стиль текста
- Простой и дружеский язык
- Без канцелярита и сложных оборотов
- Как будто объясняешь живому человеку

## Процесс работы
- Перед созданием нового файла — сначала описать что планирую сделать, получить подтверждение
- Не добавлять функции которые не просили
- Не усложнять без необходимости
