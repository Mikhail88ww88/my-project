# Mumkam.ru — План работ

## Перенос сайта на свой хостинг

- [ ] Купить VPS на Timeweb Cloud (2 CPU / 2GB RAM / 40GB SSD, ~650₽/мес)
- [ ] Настроить сервер (Ubuntu 22.04)
- [ ] SSL-сертификат (Let's Encrypt — бесплатно)
- [ ] Создать HTML главной страницы (контент и картинки уже скачаны)
- [ ] Создать страницу Москва (mumkam.ru/moskva)
- [ ] Создать страницу Санкт-Петербург (mumkam.ru/spb)
- [ ] Создать страницу Спасибо (mumkam.ru/spasibo)
- [ ] Настроить форму — заявки на почту или в Telegram
- [ ] Вставить Яндекс Метрику (ID счётчика: 99820099)
- [ ] Ночью переключить DNS на Рег.ру

## AI-продавец на сайт

- [ ] Сделать виджет чата (кнопка + окно диалога)
- [ ] Подключить Claude API
- [ ] Написать системный промпт (персонажи, цены, логика)
- [ ] Настроить логику цен (4500₽ базовая, Москва/СПб отдельно, выезд за город)
- [ ] Сбор телефона клиента
- [ ] Передача лида в AmoCRM (воронка "Горячие лиды")
- [ ] Незавершённые диалоги → AmoCRM (воронка "Не дошли")
- [ ] Уведомление в Telegram при получении телефона
- [ ] Цель в Яндекс Метрике при получении телефона
- [ ] Фиксация UTM-меток
- [ ] Показ/скрытие номера телефона по рабочему времени (06:00–20:00 МСК)

## SEO и продвижение

- [ ] Правильные мета-теги на каждой странице
- [ ] Заголовки H1-H6 с ключевыми словами
