## 3. Дизайн

**Цвета:**
- Фон: `#0a0a0a` → `#111827` (градиент)
- Текст: `#f1f5f9`
- Акцент: `#f97316` (оранжевый)
- Карточки: `rgba(255,255,255,0.05)` с `backdrop-filter: blur`
- Тени кнопок: оранжевое свечение `box-shadow: 0 0 20px rgba(249,115,22,0.4)`

**Шрифт:**
- Google Fonts: Inter (основной)
- Размер H1: 56–72px
- Размер текста: 16–18px
- Line-height: 1.6

**Настроение:** Профессиональный, современный, технологичный. Не корпоративный — живой и человечный.

## 4. Обязательные элементы

**Анимации:**
- Kinetic Typography в Hero — слово меняется каждые 2 сек
- Scroll animations — секции вплывают при скролле (IntersectionObserver)
- Oversized полупрозрачные цифры в Hero
- Hover на карточках — лёгкий подъём и оранжевая обводка
- Кнопка CTA — оранжевое свечение при hover

**Технические требования:**
- Один HTML-файл (HTML + CSS + JS внутри)
- Никаких внешних библиотек кроме Google Fonts
- Адаптив: корректно на телефоне (mobile-first)
- Мета-теги для SEO
- Яндекс Метрика (ID добавим позже)
