# Site-Bot — AI-консультант для сайта (Этап 1)

## Что делаем

Виджет онлайн-чата на сайт. AI-продавец который консультирует клиентов 24/7
и собирает номер телефона. Бизнес — поздравления ростовыми куклами.

---

## Стек

- **AI** — Claude API (Anthropic)
- **Бэкенд** — Python
- **База данных** — PostgreSQL (хранение всех диалогов)
- **Хостинг** — VPS

---

## Что умеет бот

1. Консультирует по услуге — рассказывает о персонажах и ценах
2. Работает 24/7 — не упоминает рабочие часы клиенту
3. Собирает номер телефона клиента
4. Сохраняет весь диалог в БД
5. При получении телефона — уведомление владельцу в Telegram
6. Фиксирует UTM-метки (с какой рекламы пришёл клиент)
7. Срабатывает цель в Яндекс Метрике при получении телефона

## Логика цен

- Базовая цена — 4 500₽ (все города кроме Москвы и СПб)
- Москва и СПб — цена уточняется с владельцем
- Выезд за город — доплата по км от центра (уточняется)
- Бот называет примерную цену, финальную подтверждает менеджер

## Выводы из исследования — что берём в проект

### Почему своя разработка, а не готовая платформа
Jivo, Carrot Quest, Chaport — берут подписку вечно. AI-оператор у Jivo стоит 13 000₽/мес отдельно.
Своя разработка на Claude API при 300 диалогах в месяц — ~500–2 000₽/мес. Плюс можно перепродавать другим бизнесам.

### UX — что важно из исследования
- Виджет появляется через 5–10 секунд после захода, не сразу
- Быстрые кнопки в начале: "Узнать цену", "Проверить наличие"
- **Главное правило:** сначала собираем телефон — потом обещаем перезвонить (не наоборот)
- Бот не говорит про рабочие часы — просто передаёт заявку специалисту
- Срочные запросы ("сегодня", "срочно") — немедленное уведомление владельцу

### Ключевой инсайт
Ночью уходит 30–40% лидов если бот не собирает телефон и говорит "перезвоним в рабочее время".
Решение: бот всегда предлагает оставить номер — в любое время суток.
